# Reglas del Usuario (User Rules)

Siempre que trabajemos en un proyecto, ya sea existente o nuevo, debes seguir estas instrucciones:

1. **Integración con GitHub**: En cada modificación, cambio o actualización, asegúrate de enlazar y sincronizar todo con GitHub utilizando las herramientas de MCP disponibles.
2. **Enfoque en el Proyecto Actual**: Mantén el enfoque exclusivamente en el proyecto en el que estamos trabajando en ese momento. No modifiques ni interfieras con otros proyectos o directorios ajenos a la tarea actual.
3. **Idioma**: Háblame siempre en español.
4. **Calidad de Código**: Revisa siempre los errores de lint y asegúrate de que el código sea correcto antes de dar por terminada una tarea.
