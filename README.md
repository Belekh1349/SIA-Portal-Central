# SIA - PORTAL CENTRAL

## Suite Integrada Administrativa

Este repositorio contiene la arquitectura central del portal **SIA (Sistema Integrado Administrativo)** y sus aplicaciones modulares nativas.

### Aplicaciones Incluidas
- **[Portal Central](index.html)**: Punto de acceso unificado con diseño premium.
- **[Gestión de Documentos](gestion-documentos/index.html)**: Control de entradas, salidas y formatos institucionales.
- **[Calendario de Actividades](calendario-actividades/index.html)**: Seguimiento diario con asistente de IA (Gemini).
- **[Oficios](control-oficios/index.html)**: Control y seguimiento de correspondencia oficial.
- **[Pobalines](pobalines/index.html)**: Consulta inteligente de normatividad con IA.
- **[Finanzas](finance-dashboard/index.html)**: Dashboard de control presupuestal.

### Tecnologías
- HTML5 / CSS3 (Vanilla & Tailwind)
- JavaScript (ES6+)
- Google Gemini AI Integration
- jsPDF para generación de reportes institucionales
