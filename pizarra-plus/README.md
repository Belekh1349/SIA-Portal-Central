# 🚀 Pizarra Plus - Edición Premium

Bienvenido a la versión mejorada de **Pizarra**. Esta aplicación ha sido rediseñada para ofrecer una experiencia de usuario de alto nivel, integrando inteligencia artificial y colaboración en tiempo real.

## ✨ Mejoras Clave
- **Interfaz Glassmorphism**: Diseño limpio y moderno con efectos de desenfoque.
- **Animaciones Fluidas**: Navegación suave impulsada por `framer-motion`.
- **Búsqueda Global**: Encuentra cualquier tarea, documento o comunicado al instante.
- **Asistente IA**: Integración optimizada con Gemini para ayudarte en tu flujo de trabajo.
- **Modo Administrador**: Control total sobre las publicaciones.

## 🛠 Instalación y Uso

1. **Entrar al directorio**:
   ```bash
   cd /Users/david/.gemini/antigravity/scratch/pizarra-plus
   ```

2. **Ejecutar en desarrollo**:
   ```bash
   npm run dev
   ```

3. **Configuración de Firebase**:
   Asegúrate de que la variable `__firebase_config` esté disponible en tu entorno o edita el inicio de `src/App.jsx` con tus credenciales.

## 🤖 IA Gemini
Para activar las funciones de IA, inserta tu clave de API en `src/App.jsx`:
```javascript
const apiKey = "TU_API_KEY_AQUÍ";
```
